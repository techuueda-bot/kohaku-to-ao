# 琥珀と青

架空の中華そば店を想定した、1ページ構成のブランドサイトです。

## 公開ページ

https://techuueda-bot.github.io/kohaku-to-ao/

## 概要

「青い器に、琥珀の一杯を。」という言葉を軸に、静けさ・余白・食後の余韻をWeb上で表現しました。

実在する店舗・住所・営業情報とは関係のない、ポートフォリオ練習用の架空サイトです。

## コンセプト

青い器と琥珀色の一杯を象徴に、派手な販促ではなく、一杯を食べる前後の時間を感じられるWeb体験を目指しました。

薄墨の霧による導入、縦書き文字を使った横スクロール、季節ごとに空気感を変えられる Season Note など、一般的な飲食店サイトとは違う静かな体験設計を意識しています。

## 主な特徴

- 生成りを基調にした落ち着いた画面設計
- 青い器と琥珀色の一杯を想起させる配色
- 薄墨の霧がほどけるようなオープニング演出
- 縦書きタイポグラフィを使った横スクロール演出
- 情報量を抑え、余韻を残す構成
- 架空制作物であることを明記した安全な設計

## 使用技術

- HTML
- CSS
- JavaScript
- GitHub Pages

## 制作で意識したこと

- 余白を使って静けさを作ること
- 文字を情報ではなく、景色のように見せること
- 写真に頼りすぎず、抽象的な器のグラフィックで一杯を象徴すること
- 実在店舗と誤認されないよう、架空制作物であることを明記すること

## 注意

このサイトは架空の制作物です。  
実在する店舗・住所・営業情報とは関係ありません。
