# 琥珀と青

架空の中華そば店を想定した、1ページ構成のブランドサイトです。

## 概要

「青い器に、琥珀の一杯を。」という言葉を軸に、静けさ・余白・食後の余韻をWeb上で表現することを目指しました。

実在する店舗・住所・営業情報とは関係のない、ポートフォリオ練習用の架空サイトです。

## コンセプト

青い器と琥珀色のスープを象徴に、派手な販促ではなく、一杯を食べる前後の時間を感じられるサイトを目指しています。

最初に表示される薄墨の霧は、雨の日に店の前へ立った時の空気をイメージしています。
横スクロールでは、縦書きの言葉を景色のように流し、食べる前の静けさや食後の余韻を表現しました。

## 主な構成

- 薄墨の霧によるオープニング演出
- 「琥珀と青」のヒーローセクション
- 縦書き文字を使った横スクロール思想セクション
- 二段構成のコンセプトセクション
- 静かなメニュー紹介
- 季節ごとに空気感を変える Season Note
- 架空アクセスとフッター

## こだわった点

- 写真に頼らず、器の抽象グラフィックで一杯を象徴したこと
- 横スクロールと縦書きで、言葉を情報ではなく景色として見せたこと
- ラーメン店らしい強い販促表現を避け、余白と静けさを優先したこと
- 架空制作物であることを明記し、実在店舗と誤認されない構成にしたこと

## ローカルでの確認方法

ターミナルでこのフォルダに移動し、以下を実行します。

```bash
python3 -m http.server 8000
```

Macで確認する場合:

```text
http://localhost:8000
```

同じWi-Fi内のスマホで確認する場合は、MacのIPアドレスを調べて以下のように開きます。

```text
http://MacのIPアドレス:8000
```

サーバーを止める時は、ターミナルで `control + C` を押します。

## Logo Mark

「琥珀と青」のロゴマークとして、青い弧と琥珀色の円で構成したマークを採用します。

保存場所:

- assets/logo/kohaku-to-ao-mark.png
- assets/logo/kohaku-to-ao-mark-transparent.png

`kohaku-to-ao-mark.png` は確認用の背景あり版、`kohaku-to-ao-mark-transparent.png` は実装用の透過版として管理します。

このロゴマークは、青い器、中央の琥珀、食後の余韻を抽象的に表現したものです。
サイト左上の小さなロゴマーク、favicon候補、README、ポートフォリオ掲載用素材として使用予定です。

現時点ではPNG画像として管理し、必要に応じて将来的にSVG化を検討します。

### 使用方針

- ロゴマークは小さく使う
- ヒーローの主役にはしない
- 「琥珀と青」の文字見出しと主役争いさせない
- 器の抽象グラフィックとは役割を分ける
- ロゴアニメーションは後工程として扱う
- 回転、大きな揺れ、派手な波紋は避ける
