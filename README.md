# 琥珀と青

架空の中華そば店を題材にした、1ページ構成のポートフォリオ用ブランドサイトです。

公開ページ: https://techuueda-bot.github.io/kohaku-to-ao/

## リニューアルの狙い

従来の「静けさ・余白・縦書き」の世界観を残しつつ、次の点を強化しました。

- ファーストビューで中華そば店だと伝わる料理写真と明確なコピー
- どこからでも予約デモへ進めるナビゲーションとスマホ固定CTA
- 写真付きのお品書き、営業時間、アクセス、FAQ
- 長すぎた横スクロールを短縮し、スマホは横スワイプに切り替え
- 入力内容を外部へ送信しない予約フォームの操作デモ
- OGP、canonical、レスポンシブ、キーボード操作への対応

## 重要: 架空サイトです

店舗名、住所、営業時間、料金、メニュー、席数はすべて架空です。予約フォームに入力した内容は、保存も送信もされません。

実案件で使う場合は、フォーム送信先と実店舗情報を必ず差し替えてください。

## ファイル構成

- `index.html`: ページ本文、OGP、予約デモフォーム
- `style.css`: 配色、レイアウト、レスポンシブ、モーション
- `script.js`: ナビ開閉、スクロール演出、フォーム検証
- `assets/images/_original/hero-chuka-soba.png`: AI生成した料理写真の原本
- `assets/images/hero-chuka-soba-1536.webp`: PC向け画像
- `assets/images/hero-chuka-soba-960.webp`: スマホ向け画像
- `assets/images/ogp-kohaku-to-ao.jpg`: SNSシェア用画像
- `assets/logo/kohaku-mark-256.webp`: 表示用の軽量ロゴ
- `assets/logo/favicon-64.png`: 表示用の軽量favicon

## ローカル確認

リポジトリのフォルダで次を実行し、表示されたURLをブラウザで開きます。

```bash
python3 -m http.server 8000
```

確認URL: `http://localhost:8000/`

## 変更するときの主な場所

- 配色: `style.css` 冒頭の `:root`
- メニュー・価格: `index.html` の `#menu`
- 営業情報: `index.html` の `#access`
- 季節の一杯: `index.html` の `#season`
- 予約フォームの動作: `script.js` の `data-demo-form` 周辺

## 実運用フォームへの変更

GitHub Pagesにはサーバー側の送信機能がありません。実運用する場合は、Formspree等で利用者本人が送信先を作成し、`form` の `action` を差し替えてください。公開後に必ずテスト送信し、通知先・自動返信・スパム対策を確認します。

## 外部依存

日本語書体としてGoogle Fontsを読み込んでいます。オフライン環境ではシステムの明朝・ゴシック体へフォールバックします。

## AI生成画像

組み込み画像生成を使用し、次の内容で料理写真を作成しました。

> 深い藍色の器に入った、澄んだ琥珀色の醤油中華そば。雨上がりの静かなカウンター、自然な食材と陶器の質感、青い影と温かな出汁の光を対比した横長の料理写真。文字・ロゴ・人物なし。

画像の原本は `_original` に保全し、Web表示用画像は原本から一度だけWebPへ書き出しています。

## 公開前チェック

- 架空の住所・営業時間・料金を使う場合は、架空サイト表記を残す
- 実案件化する場合は、店舗情報と予約送信先を実データへ差し替える
- `og:url`、`og:image`、canonicalが本番URLと一致しているか確認する
- 予約フォームを実送信型に変えた場合は、プライバシーポリシーを用意する
- 公開後にPC・スマホで主要導線とコンソールエラーを確認する

## 公開後の改善候補

実案件では、予約CTAのクリック数・予約完了率・流入元を計測します。最初の改善候補は、ファーストビューのコピーと予約ボタン文言の比較です。公開3か月後にメニュー閲覧率と予約到達率を見直します。
